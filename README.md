Project Title: HTTP-Traffic-Analysis-with-Wireshark
objectives: This project analyzes HTTP traffic captured using wireshark.
Tools: Wireshark
Methodology: Captured network traffic, Applied HTTP filter, and Investigated HTTP GET request and response code
Findings: Observed HTTP GET request and response code
Security Insight: HTTP is not encrypted, attackers can see URLs, usernames, passwords and cookies.
Skills Demostrated: Network traffic analysis, Packet inspection HTTP investigation
